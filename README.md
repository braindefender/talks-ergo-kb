# Talks / Ergonomic Keyboards

Презентация об эргономичных клавиатурах

В ней я рассказываю про:
- Эргономику рабочего места и об эргономике в целом.
- Проблемы современных клавиатур и как их решают.
- Что такое эргономичные клавиатуры и с чем их едят.
- Как уместить 104 клавиши в 36 клавиш на примере Wellum.

Была рассказана:

[2025-11-25] TE React Созвон
[2025-11-28] TE Smart Meetup
